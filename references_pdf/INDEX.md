## 命名规则

所有 PDF 文件按照以下命名规则：

```
<作者姓氏><年份><关键词>.pdf
```

例如：
- `openai2023gpt4.pdf` → OpenAI 2023 GPT-4
- `touvron2023llama.pdf` → Touvron 2023 LLaMA
- `feng2020codebert.pdf` → Feng 2020 CodeBERT

这样可以与 `ref/refs.bib` 中的 BibTeX 引用 key 保持一致。

---

## 使用说明

1. **在 LaTeX 中引用**：使用 `ref/refs.bib` 中的 BibTeX key
2. **查找 PDF**：根据 BibTeX key 对应的文件名在 `references_pdf/` 目录查找
3. **添加新文献**：下载后按照命名规则重命名，并更新本索引文档
